###Contact Us
We are interested in help with any of the following:

* Documentation
* R examples
* Core code development

We follow the [fork+pull](http://help.github.com/send-pull-requests/) collaboration model.
Please add an issue to the issue tracker or comment on an existing one to describe what you intend to do. This is to stimulate community input and increase the chances of your patch being accepted.

For any questions, feel free to reach us at rhadoop@revolutionanalytics.com.
